Description
A web-based Hospital Management System for managing hospital activities, including:

Online appointment system

User-level wise reports

Doctor scheduling management

SMS system integration (for notifications)

Patient management system

This system is built using HTML, CSS, JavaScript, and PHP, with database connectivity for managing hospital data efficiently.

Features
View and manage different hospital departments (Surgery, Diagnostics, Radiology, etc.)

Patient registration and admission management

Disease and treatment tracking (COVID-19, Tuberculosis, Cancer, etc.)

Doctor and staff scheduling

User authentication for different roles (admin, principal, faculty, office staff, students)

Centralized dashboard for quick hospital summary

Feedback and grievance module

How To Run
Place all HTML, PHP, JS, and CSS files in your web server directory (e.g., htdocs for XAMPP).

Set up MySQL database named patient and run the relevant table/record creation scripts as described in patient1.php.

Start your Apache and MySQL services (XAMPP/LAMP/MAMP recommended).

Open index1.html in a browser or go to localhost/path-to-your-folder/index1.html.

Navigate through the menu to explore features: Patient registration, department info, diseases, etc.
